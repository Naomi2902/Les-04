Stappenplan;
html & CSS

Stap 1

Open in Visual Studio Code een nieuw bestand. Op de onderstaande foto kan je zien met welke tags je moet beginnen. Sla het bestand op als een .html bestand.

Stap 2

In de head kan je een titel toevoegen, deze titel is terug te vinden in de tablad. Je kan ook eventueel nog een icoontje toevoegen. In de body tag doe je alle informatie, foto's en div tags in. Met div tags maak je groepen. In de footer kan je een copyright met je naam invoegen.

Stap 3

Een html zorgt voor de structuur van je pagina, en css is voor de opmaak van je pagina. Je maakt een nieuw bestand aan en die sla je op als .css. Daarna ga je in de head je html en css linken met elkaar.

Stap 4

In html maak je een div id voor elke div die je hebt in je html. Deze geef je namen en met css kan je deze nu gaan opmaken. Je kan bepalen wat de positie is van de tekst/foto's en wat voor font en kleur de tekst krijgt
